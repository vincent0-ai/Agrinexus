## 🛠️ How to Use These in Figma

1. Open **Figma** → New File
2. Install plugin: **Figma AI** or **Galileo AI** or **Magician**
3. Paste each prompt into the AI design tool
4. Generate → adjust colors to match: `#2D6A4F` (dark green), `#F4A261` (amber), `#FFFFFF`
5. Build a **component library** first (buttons, cards, nav) then apply across screens
